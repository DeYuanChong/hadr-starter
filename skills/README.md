# Skills

Optional [Claude Code skills](https://code.claude.com/docs/en/skills) for this
repository. Each skill lives in its own folder as a `SKILL.md` plus any
supporting assets.

Skills are a development convenience only — they are **not** required to build,
test, deploy, or run the HADR Monitor. The situation report is produced by the
deterministic pipeline in [`src/`](../src/) (`npm run build:dashboard`).
