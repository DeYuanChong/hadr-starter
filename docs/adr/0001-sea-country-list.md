# Southeast Asia scope is ASEAN-10 plus Timor-Leste, nothing more

We need a fixed definition of "Southeast Asia" to filter three global
feeds. We chose the ASEAN ten (Brunei, Cambodia, Indonesia, Laos, Malaysia,
Myanmar, Philippines, Singapore, Thailand, Vietnam) plus Timor-Leste, and
explicitly excluded Papua New Guinea and southern China even though both
border in-scope countries. Geographic or seismic proximity was rejected as
a basis for inclusion — PNG is Melanesia, not Southeast Asia, by every
standard geoscheme and by ASEAN membership, and including border regions
of China would blur the regional filter this project exists to provide. If
an in-scope country is independently affected by a cross-border event
(e.g. a Yunnan earthquake felt in northern Vietnam), that event still
surfaces via the in-scope country's own data, not via a broadened country
list.
