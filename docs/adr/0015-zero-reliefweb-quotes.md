# Zero direct quotes from ReliefWeb content, own-words summary only

ReliefWeb content belongs to ~4,000 publishers; the site's T&C forbids
redistribution and derivative works, and now includes an explicit clause
about redistributing AI-generated content attributed to sources.
`blindspots.md` flags the common assumption that "metadata is fine, full
text isn't" as community convention, not a written grant.

We chose zero direct quotes, not "quote sparingly" as REQS.md initially
put it. Any ReliefWeb-derived content in the sitrep is a one-sentence,
own-words summary (an Own-words summary) plus attribution and a link back
— never a copied sentence or phrase from the source.

Given the stated legal ambiguity plus an AI-specific clause the project
falls squarely under, zero quotes is a materially safer line to draw than
any nonzero "sparingly" threshold, and is simpler to implement and review
than a fuzzy quoting budget. Since ReliefWeb links are already
supplementary and non-blocking to the report's core purpose (ADR-0004,
ADR-0009), no case in this build depends on richer quoted content.
